## ğŸš€ Overview

**DIS (Data Integrity & Security)** is a multi-module encryption project that offers real-time encryption and decryption of:
- Text
- Images (using pixel-level manipulation)
- Files (PDF, TXT, DOCX, etc.)

Each module is built using Python and Flask, making them lightweight and interactive through a local web server.

## âš™ï¸ How to Run

```bash
git clone https://github.com/Madhu1207-coder/DIS.git
cd DIS
python -m venv venv
source venv/bin/activate  # For Windows: venv\Scripts\activate
pip install -r requirements.txt
````

### Run Each App

```bash
# Module 1: Text Encryption
cd Text_Encryption
python text_app.py

# Module 2: Image Encryption
cd ../Image_Encryption
python image_app.py

# Module 3: File Encryption
cd ../File_Encryption
python file_app.py
```

Navigate to `http://127.0.0.1:5000` in your browser for each module.
